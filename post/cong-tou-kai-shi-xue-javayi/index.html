在搭建之前 必不可少的是要先配置好语言的环境：

IDE

对比了一下之后我选择了 IntelliJ IDEA，别问，问就是JetBrains脑残粉！

Java

Java版本我选择14.0.2，这个可以自行选择下载。

在下载了IntelliJ IDEA社区版的最新版本之后我发现其中并没有直接包含Spring Initializr，而是要我们自己去寻找。

在File->Settings->Plugins->MarketPlace里面找到Spring Assistant点击下载即可。
